Elementos de trabajo

Desde este menú es posible crear, modificar y borrar los Elementos de trabajo de la organización. Estos elementos pueden ser asociados a puestos de trabajo y puede registrarse su entrega, su devolución, su pérdida o rotura, etc.

 Entrega periódica

Permite establecer que el elemento se entrega periódicamente, es decir, cada un cierto intervalo de tiempo.

Esta información se utiliza luego para establecer, mediante informes, qué elementos de trabajo serán necesarios entregar a una determinada fecha.

Cada meses

Permite establecer la entrega periódica cada cierta cantidad de meses, independientemente del mes en cuestión. Solo acepta números enteros.

Los meses:

Permite establecer uno o más meses puntuales para la entrega del elemento. El sistema asume que los elementos se entregarán el primer día de dichos meses.

 Requiere identificación

Si está activado, se solicitará la identificación del elemento cuando se cargue una  novedad de entrega, devolución y/o pérdida/rotura.

Sirve para realizar una traza del elemento en cuestión.

 Requiere devolución

Permite establecer que el elemento requiere ser devuelto. El sistema realiza controles sobre elementos no devueltos al cargar novedades de entrega y al cargar novedades de egreso de personal. Además, existe un informe de elementos no devueltos.

 Posee certificación

Permite establecer que el elemento posee certificación de algún ente.

 Puestos

Permite asociar el elemento de trabajo a uno o varios puestos de trabajo. Es posible, además, establecer la cantidad requerida en cada puesto.

 Capacitaciones

Permite asociar las capacitaciones requeridas para el uso del elemento de trabajo, pudiendo establecerlas como obligatorias o no. En caso de ser obligatorias, el sistema controla que las mismas hayan sido informadas al momento de la carga de una novedad de entrega; impidiendo la carga en caso de no existir constancia de la realización de dichas capacitaciones.

 Atributos

Permite asociar atributos de elementos de trabajo.
