Cambio de empresa

Esta tarea permite ingresar a otra empresa de las que se encuentran habilitadas para el usuario.

Si se presiona el botón se exhibe una pantalla con las empresas disponibles.

 Usuario

Exhibe el nombre de usuario que está operando el sistema.

 Empresa

Exhibe el código de la empresa actual.

Se debe ingresar el código correspondiente a la empresa en la cual se quiere operar.

Puede utilizarse el botón de búsqueda para acceder a la ventana de búsqueda y selección de empresas.
