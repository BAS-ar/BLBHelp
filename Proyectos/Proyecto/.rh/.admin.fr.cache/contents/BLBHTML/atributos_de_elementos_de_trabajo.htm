Atributos de Elementos de trabajo Los atributos de Elementos de trabajo permiten agrupar por determinadas características a los elementos de trabajo. Luego, pueden ser utilizados en la emisión de los informes de elementos de trabajo.  Código  Valor alfanumérico asignado para identificar el atributo seleccionado. Ej.:(E01).  Descripción Descripción asignada al atributo. Ej (Elemento de protección).  Valores Valores que puede tomar el atributo.  Ej.:(Si, No).
