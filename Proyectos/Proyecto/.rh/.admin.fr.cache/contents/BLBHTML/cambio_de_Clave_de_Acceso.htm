Cambio de Clave de Acceso Esta tarea permite modificar la clave de acceso del usuario que está operando el sistema.  Usuario Exhibe el nombre de usuario que está operando el sistema.  Claves de Acceso Permite modificar la clave del usuario.  Anterior Ingresar la clave actualmente en uso. A medida que se ingresa se exhiben asteriscos.  Nueva Ingresar la nueva clave. A medida que se ingresa se exhiben asteriscos.  Confirmación Reingresar la clave informada en "Nueva". A medida que se ingresa se exhiben asteriscos.  Para tener en cuenta: Cada usuario puede modificar su clave libremente las veces que desee  Temas relacionados Usuarios
