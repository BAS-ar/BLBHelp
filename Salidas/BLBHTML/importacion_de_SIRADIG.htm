Importación de SIRADIG

Para facilitar la carga de las novedades de ganancias, es posible importar las novedades del SIRADIG. Este punto del menú permite importar los datos informados en el Formulario 572 Web (SIRADIG).

 Resultado

Muestra la información de las presentaciones (la cantidad de presentaciones leídas correctamente y la cantidad de presentaciones importadas correctamente y con error).

 Opciones

Respetar secuencia de presentaciones

Si se activa este campo, la secuencia de las presentaciones de cada empleado deberá ser consecutiva.

Mostrar detalle del procesos

Al activar este campo, se mostrará el detalle de cada novedad importada.

 Carpeta a importar

 Archivo a importar

 Se indica la ruta de acceso al archivo de texto a importar.

 Aceptar

Para comenzar la importación necesario presionar este botón,..

 Cancelar

Limpia la pantalla de datos sin registrar los datos ingresados.

 Para tener en cuenta:

Con anterioridad al proceso de importación, deberán generarse, desde la Web de la Agencia de recaudación nacional , los archivos que se desean importar. Estos archivos deberán tener extensión .XML y deben guardarse en una carpeta destinada a tal efecto.

Se aconseja utilizar la ruta predeterminada para Windows 7 (y posteriores) ya que no se presentarán inconvenientes con los permisos.

Las novedades de ganancias informadas mediante importación de SIRADIG, no se pueden modificar.
